Physical layer implementations for CAN and CAN-FD Physical layer implementations for automotive protocols.

This module provides implementations for the physical layer protocols:

  • Classic CAN (Controller Area Network)
  • CAN-FD (CAN with Flexible Data-Rate)

The physical layer is responsible for the actual transmission and reception of bits on the physical medium. It handles:

  • Bit timing and synchronization
  • Signal levels and electrical characteristics
  • Frame formatting and bit stuffing
  • Error detection at the physical level
